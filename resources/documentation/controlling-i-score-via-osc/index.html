This page is still under writing. In the mean time, here is a rough reference of OSC messages that can be used to remotely control i-score.

/Main:Start: Launch i-score scenario execution
/Main:Pause: Pause i-score scenario execution
/Main:Resume: Resume i-score scenario execution
/Main:End: End i-score scenario execution
/Main:Goto: Set start time of scenario, in milliseconds
/Main:speed: Set speed of the scenario execution
/Main:Next (optionnal list): Trigger the next interactive automation box (if no argument provided) or the list of numbered trigger point

/'box name':Start: Start automation box (named ‘box name’)
/'box name':Pause: Pause automation box (named ‘box name’)
/'box name':Resume: Resume automation box (named ‘box name’)
/'box name':End: End automation box (named ‘box name’)
