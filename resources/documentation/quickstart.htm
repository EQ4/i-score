This quick start gives a brief overview of i-score workflow so you can start writing your own scenario.

Adding devices

Applications that you want to control from i-score need to be declared as a Device (a distant software or hardware) so you can address its parameters.

  • Add OSC device

    In the Device explorer, click the "Add a device" button to bring the device settings popup window. Then set the name of the application, choose OSC from the protocols menu and set the different network informations
    Then use one of the following options

    • Load namespace file of your OSC application using the "Load" button from the device setting window
    • After expanding the device namespace using the triangle button, click the "Add a node" button to manually add an OSC address
    • Use OSC learn mode by clicking the checkbox on the left of your device name in the Device explorer panel
