Creation

 Boxes can be created by clicking and releasing the mouse
at the wished dimensions of the box.

Edition


: Copy : Ctrl/Cmd+C : Copy a selection of boxes
: Cut : Ctrl/Cmd+X : Cut a selection of boxes
: Paste : Ctrl/Cmd+V : Paste a selection of boxes

View


: Zoom In : Ctrl/Cmd+'+' : Process a Zoom In
: Zoom Out : Ctrl/Cmd+'-' : Process a Zoom Out
: Explorer : Ctrl/Cmd+E : Open/close device explorer
For further information about the devices explorer, see "Explorer" tab.

Acceleration



: Speed : No Shortcut : Sets the speed of the scenario's execution

Goto



: Time Bar : No Shortcut : Clicking in the Time Bar will set the starting point of the scenario the next time it is played
